МЕДЬ (лат. Cuprum)

Общие сведения

Химический элемент таблицы Менделеева, металл.
Символ элемента: Cu.
Атомный номер: 29.
Положение в таблице: 4-й период, группа - IВ (11).
Относительная атомная масса: .
Степени окисления (жирным шрифтом выделены наиболее характерные): +1,+2,.
Валентности (жирным шрифтом выделена наиболее характерная): ,+3 и +4.
Электроотрицательность: 1,9.
Электронная конфигурация: [Ar]3s2p6d104s1.
Природная медь состоит из стабильных нуклидов : 63Cu (69,09% по массе) и 65Cu (30,91%). В ряду стандартных потенциалов медь расположена правее водорода и ни из воды, ни из кислот водорода не вытесняет. Простое вещество медь — красивый розовато-красный пластичный металл.
